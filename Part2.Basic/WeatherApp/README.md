## URLSession Life Cycle

    1. Session Configuration을 결정하고, Session을 생성
    2. 통신할 URL과 Request 객체를 설정
    3. 사용할 Task를 결정하고 그에 맞는 Completion Handler나 Delegate 매소드들을 작성
    4. 해당 Task를 실행
    5. Task 완료 후 Completion Handler 클로자가 호출이 됨
